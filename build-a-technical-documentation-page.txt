      <header>Variables and Data Types</header>
      <p>Variables are containers for storing data values.</p>
      <code>let age = 30;</code>
      <p>Data types include strings, numbers, booleans, and more.</p>
      <ul>
        <li>String</li>
        <li>Number</li>
        <li>Boolean</li>
      </ul>
      <p>They help you structure your program effectively.</p>
      <code>const name = "Waldo";</code>
    </section>

    <section class="main-section" id="Control_Flow">
      <header>Control Flow</header>
      <p>This refers to the order in which individual statements are executed.</p>
      <code>if (true) { console.log("Yes!"); }</code>
      <p>Conditional statements let your program make decisions.</p>
      <p>Loops allow repetition based on conditions.</p>
      <ul>
        <li>if / else</li>
        <li>switch</li>
        <li>for / while loops</li>
      </ul>
    </section>

    <section class="main-section" id="Functions">
      <header>Functions</header>
      <p>Functions group code into reusable blocks.</p>
      <code>function greet() { return "Hi!"; }</code>
      <p>They help reduce repetition and increase modularity.</p>
      <p>You can pass data into functions using parameters.</p>
      <code>greet("Waldo");</code>
    </section>

        <li>Log output to console</li>
        <li>Read error messages carefully</li>
