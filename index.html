A quick, practical guide to core concepts you’ll use every day.

Introduction

Programming is the craft of giving computers clear, testable instructions. This guide walks through core ideas you’ll encounter in any language.

Whether you’re starting out or refreshing fundamentals, these sections focus on concepts that transfer across ecosystems.

print("Hello, world!")

“Hello, world!” is a traditional first step: it checks your toolchain and output.

Variables & Data Types

Variables store values you’ll reuse or transform. Good names improve readability and maintenance.

let age = 30;

Common data types:

  • String — text values: "Waldo"
  • Number — integers or floats: 42, 3.14
  • Boolean — truth values: true or false

Use const for values that don’t change; prefer let when reassignment is required.

const name = "Waldo";

Control Flow

Control flow is the order in which statements run. Conditionals branch execution; loops repeat work.

if (Math.random() > 0.5) {
  console.log("Heads");
} else {
  console.log("Tails");
}
  • Conditionals: if, else, switch
  • Loops: for, while, for...of

Functions

Functions package logic into reusable units with clear inputs and outputs.

function greet(name = "friend") {
  return `Hi, ${name}!`;
}

Parameters pass data in; return values pass results out.

greet("Waldo"); // "Hi, Waldo!"

Debugging Tips

Debugging is systematic problem-solving. Isolate, measure, and verify.

  • Set breakpoints to inspect state and step through code.
  • Use console.log (sparingly) to trace values.
  • Read error messages carefully; they often tell you exactly where to look.
console.log("Reached point A", { value, index });

Patience and a repeatable process are your best tools.
